1 mysql> system ls /tmp(可以在mysql命令行中输入系统命令)

2 登录-p时候一定不要输入密码，否则会被history记录
或者（输入命令之前加个空格，然后设置histcontrol）
echo “export HISTCONTROL=ignorespace” >> /etc/profile                
source /etc/profile                                    


3 killall，kill，pkill命令的区别（https://www.cnblogs.com/rsky/p/4886043.html）

4 当mysql -uroot -p -U登录时候，当发出没有where或者limit的updata或者delete时候，mysql程序就会拒绝执行。（update table set name=‘gongli’）


5 更改系统语言，cat /etc/sysconfig/i18n
#LANG = “zh_CN.GB2312”
LANG = "zh_CN.utf8"

6 mysql -e，可以不用进入mysql，直接输出结果
mysql -uroot -p123456 -e "show character set";
mysql -uroot -p123456 -e "show character set；set names Latin1";
