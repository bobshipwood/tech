git merge
     a--b---c    topic			
    /
d--e--f---g           master

在master分支下，执行git merge topic命令后，在两个分支内产生一个新节点


     a--b---c    topic			
    /        \
d--e--f---g----h           master



git rebase

d----e---f      master    指向f
      \	
       g---h     feature  指向h


在feature分支下，执行git rebase master,将会变成一条直线


d---e---f---g`---h`   master指向f，  feature指向h`
